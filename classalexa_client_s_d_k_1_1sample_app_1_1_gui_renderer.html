Detailed Description

A simple class that implments the TemplateRuntimeObserverInterface. Instead of rendering the display cards, this class will print out some useful information (e.g. JSON payload) when the renderTemplateCard or renderPlayerInfoCard callbacks are called.

Note
Due to the payload in RenderTemplate may contain sensitive information, the payload will only be printed if ACSDK_EMIT_SENSITIVE_LOGS is ON.
