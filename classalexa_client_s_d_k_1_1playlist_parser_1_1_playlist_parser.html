Detailed Description

This implementation of the PlaylistParser uses the Totem Playlist Parser library.

See also
https://github.com/GNOME/totem-pl-parser
https://developer.gnome.org/totem-pl-parser/stable/TotemPlParser.html.
Note
For the playlist parser to function properly, a main event loop needs to be running. If there is no main event loop the signals related to parsing playlist will not be called.
