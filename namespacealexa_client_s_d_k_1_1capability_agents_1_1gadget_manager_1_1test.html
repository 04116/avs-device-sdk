static void alexaClientSDK::capabilityAgents::gadgetManager::test::checkEscapedByteChecksum ( uint16_t  expectedChecksum)
static

This function is for testing checksums that have at least one escaped byte. A packet is created and all of the 3 or 4 byte checksums are substituted into the Gadget Protocol packet (depending on whether the expected checksum needs 1 or 2 escaped bytes). The test checks to see wheter or not the packet is valid with the substituted checksum. If the substituted checksum is the same as the expected checksum, the packet should be valid, otherwise it is invalid.
