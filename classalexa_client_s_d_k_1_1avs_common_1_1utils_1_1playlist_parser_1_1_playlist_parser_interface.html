virtual bool alexaClientSDK::avsCommon::utils::playlistParser::PlaylistParserInterface::parsePlaylist ( const std::string &  url,
std::shared_ptr< PlaylistParserObserverInterface >  observer 
)
pure virtual

This function returns immediately. It parses the playlist specified in the url asynchronously. The playlist parsing is a DFS parsing. If the playlist contains a link to another playlist, then it will proceed to parse that before proceeding. The result of parsing the playlist is notified to the PlaylistParserObserverInterface via the onPlaylistParsed call.

Parameters
urlThe url of the playlist to be parsed.
observerThe observer to be notified of when playlist parsing is complete.
Returns
true if it was successful in adding a new playlist parsing request to the queue else false.
