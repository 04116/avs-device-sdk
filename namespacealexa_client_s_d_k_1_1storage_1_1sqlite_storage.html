◆ openSQLiteDatabaseHelper()

static sqlite3* alexaClientSDK::storage::sqliteStorage::openSQLiteDatabaseHelper ( const std::string &  filePath,
int  sqliteFlags 
)
static

A utility function to open or create a SQLite database, depending on the flags being passed in. The possible flags defined by SQLite for this operation are as follows:

SQLITE_OPEN_READONLY SQLITE_OPEN_READWRITE SQLITE_OPEN_READWRITE | SQLITE_OPEN_CREATE

The meanings of these flags are as one might expect, however for further details please refer to the online documentation here:

https://sqlite.org/c3ref/open.html

Parameters
filePathThe path, including file name, to where the database is to be opened or created.
sqliteFlagsFlags which will be passed to the SQLite call. These flags determine the method of opening.
[out]dbHandleA double-pointer to a sqlite3 handle which will be updated to point to a valid handle.
Returns
Whether the operation was successful.
